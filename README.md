# M5Cardputer Voxel Engine

A WIP voxel engine for the M5Cardputer with dynamic terrain, physics, and interactive menus.

---

## Features

- Dynamic world generation
- Procedural terrain with customizable parameters
- Multiple block types (grass, dirt, stone, water, wood, ores)
- Physics (gravity, jumping, flying, collision detection)
- Interactive menu system
- Real-time performance monitoring (FPS, face count, frametime)
- Wireframe and pointcloud rendering modes
- Day/night cycle with dynamic sky and lighting
- Configurable render distance, fog, and LOD
- Flight mode and movement controls
- Chunk-based world loading
- Performance graphs and system info

---

## Hardware Requirements

- M5Cardputer (with keyboard and display)
- ESP32

---

## Installation

1. Clone or download the repository.
2. Open `M5Cardputer_VoxelEngine.ino` in the Arduino IDE.
3. Install the M5Cardputer library.
4. Upload the sketch to your device.
